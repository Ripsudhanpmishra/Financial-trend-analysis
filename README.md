# Financial Trend Analysis & Customer Churn Dashboard

## 📌 Project Overview
This project provides an end-to-end data analytics solution for a retail financial dataset. It transforms raw, uncleaned transaction data into a strategic forecasting tool. The pipeline involves data cleaning via **Python**, relational storage in **MySQL**, and interactive visualization in **Power BI**.

### 🚀 Key Highlights
* **Data Integrity:** Achieved 99% data integrity by handling outliers and missing values using Python (Pandas).
* **Strategic Metrics:** Calculated Quarterly Revenue Growth and Customer Churn Rates using advanced SQL queries.
* **Interactive Visualization:** Developed a Power BI dashboard to enable data-driven decision-making.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x (Pandas, NumPy, SQLAlchemy)
* **Database:** MySQL 8.0
* **Visualization:** Power BI Desktop
* **Environment:** Windows (ThinkPad T470 setup)
